# CreoNow 浅色主题 Variant 设计指令

> 以下 prompt 用于让 Variant 生成浅色主题版本的设计稿。
> 每个 prompt 对应一个现有的深色设计稿。

---

## 通用浅色主题规范

在每个 prompt 开头添加以下规范：

```
设计规范（浅色主题）：
- 背景：主背景 #ffffff，表面 #f8f8f8，悬停 #f0f0f0
- 文字：主要 #1a1a1a，次要 #666666，最弱 #999999
- 边框：#e0e0e0，悬停 #d0d0d0，聚焦 #c0c0c0
- 字体：Inter（UI），Lora（正文），JetBrains Mono（代码）
- 圆角：按钮 100px（pill），卡片 24px，输入框 4px
- 动效：cubic-bezier(0.2, 0.0, 0.2, 1)，200ms
- 阴影：使用浅色阴影 rgba(0,0,0,0.08)
```

## 使用说明

1. 将每个 prompt 发送给 Variant
2. 生成的 HTML 文件保存到 `design/Variant/designs/` 目录
3. 命名格式：`XX-name-light.html`（如 `01-login-light.html`）
4. 生成后检查配色是否符合规范
